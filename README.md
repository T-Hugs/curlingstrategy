# Curling Strategy Board

A virtual curling strategy board for the browser with accurate WCF dimensions.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure the port (optional):
   - Edit `.env` file to change the `PORT` value
   - Default port is 3000

## Running the Server

### Development Mode (with hot reload):
```bash
npm run dev
```

### Production Mode:
```bash
npm run build
npm start
```

The server will run at `http://localhost:3000` (or the port specified in `.env`).
